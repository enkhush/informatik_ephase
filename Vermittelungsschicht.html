Das IP

IP

Das IP ist ein weit verbreitetes Netzwerkprotokoll und ist auch eine Grundlage des Internets. Computer können innerhalb eines Netzwerks in logische Einheiten gruppiert werden mittels IP-Adresse und Subnetzmaske; eine Subnetzmaske/Bitmaske gibt an wie viele Bits bei einer IP-Adresse das Netzpräfix ausmacht, jedenfalls in IPv4-Netzen. Diese Einheiten werden Subnetze genannt und auf dieser Ebene kann man Computer in größeren Netzwerken adressieren und IP-Pakete zu schicken lassen. Zudem ist die logische Adressierung eine Grundlage für Routing.
